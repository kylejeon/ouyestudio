
# 오우예 셀프스튜디오 사진 편집기

셀프 사진 편집 및 인화를 위한 온라인 도구입니다. 다양한 레이아웃으로 사진을 배치하고 텍스트를 추가하여 나만의 사진을 만들어보세요.

## 기능

- 📸 사진 업로드 및 편집
- 🎨 다양한 레이아웃 선택
- ✏️ 텍스트 추가 및 폰트 변경
- 💾 편집된 사진 다운로드

## 기술 스택

- React 19
- TypeScript
- Vite
- TailwindCSS
- React Router DOM

## 로컬 개발

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## GitHub Pages 배포

1. GitHub 저장소 생성
2. 소스코드 업로드
3. Settings > Pages에서 Source를 "GitHub Actions"로 설정
4. main 브랜치에 푸시하면 자동 배포
